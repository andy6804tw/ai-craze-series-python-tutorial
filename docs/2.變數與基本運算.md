# 2.變數與基本運算
## Basic Syntax
## Variables
```py
# assignment
a = 1
b = c = 5

# assign multiple objects to multiple variables.
a, b, c = 1, 2, "John"
print(a) # 1
print(b) # 2
print(c) # John
```

## Data Type
```py
counter = 100 # An integer assignment
miles = 1000.0 # A floating point
name = "John" # A string
print(counter) # 100
print(miles) # 1000.0
print(name) # John
```

## Keywords
保留字

## Arithmetic Operators
```py
add = 1 + 1 # 2
sub = 1 - 1 # 0
div = 4 / 2 # 2
mod = 4 % 3 # 1
mul = 2 * 3 # 6
f_div = 5 // 2 # 2
power = 2 ** 3 # 8
```

## Comparison Operators
```py
a, b = 10, 20

a == b # False
a != b # True
a < b # True
a > b # False
a <= b # True
a >= b # False
```

## Built-in Functions
e.g. print( ), type( ), int( ) and str( )
```py
integer = 123
string = "456"

s_to_i = int(string) # int now
i_to_s = str(integer) # str now
print(type(s_to_i)) # <class 'int'>
print(type(i_to_s)) # <class 'str'>
```
